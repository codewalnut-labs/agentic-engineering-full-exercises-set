# Exercise 02 : Mermaid C4 and Sequence Diagrams

## Your Mission

Your mission is to reverse engineer the workflow and create diagrams that match the actual code paths.

You are given a repository with no trustworthy diagrams and several non-obvious runtime interactions.

The duration for this challenge is 30 min or less.

## Project

[workflow-diagram-app](./workflow-diagram-app) contains the workflow diagram source for this exercise.

## How To Go About It

Use [Mermaid](https://mermaid.js.org/) to create C4-style and sequence diagrams.

Ask your coding agent to inspect `workflow-diagram-app/`, trace the workflow, create the diagrams, and verify them against the code.

## Evidence

Produce Mermaid diagram files and verification notes.

Raise the completed work as a PR for getting verified with our team.
