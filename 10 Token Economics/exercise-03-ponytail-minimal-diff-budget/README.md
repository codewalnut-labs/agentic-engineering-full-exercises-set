# Exercise 03 : Ponytail Minimal-Diff Budget

## Your Mission

Your mission is to complete a migration slice with the smallest safe diff.

You are given a repository with a design-system migration that invites broad edits and unnecessary token spend.

The duration for this challenge is 30 min or less.

## Project

[minimal-diff-app](./minimal-diff-app) contains the minimal-diff workflow for this exercise.

## How To Go About It

Use the [Ponytail](https://github.com/DietrichGebert/ponytail/blob/main/skills/ponytail/SKILL.md) minimal-diff skill workflow.

Ask your coding agent to inspect `minimal-diff-app/`, choose one migration slice, keep the diff minimal, and verify behavior.

## Evidence

Produce the minimal diff, avoided-work notes, budget notes, and verification output.

Raise the completed work as a PR for getting verified with our team.
