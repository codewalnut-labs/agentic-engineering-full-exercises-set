# Exercise 02 : Context Cache Hygiene Challenge

## Your Mission

Your mission is to remove stale always-on context and replace it with lean, useful instructions.

You are given a repository where outdated agent notes cause repeated wrong edits and unnecessary context loading.

The duration for this challenge is 30 min or less.

## Project

[context-cache-app](./context-cache-app) contains the context cache workflow for this exercise.

## How To Go About It

Use [Context7 documentation](https://context7.com/docs/overview) or current project docs only when they affect the task.

Ask your coding agent to inspect `context-cache-app/`, prune stale context, complete one small change, and verify future-agent instructions.

## Evidence

Produce the cleaned context files, working change, and verification output.

Raise the completed work as a PR for getting verified with our team.
