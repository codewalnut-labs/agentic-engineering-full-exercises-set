# Exercise 03 : OpenAI Evals Model Routing Planner

## Your Mission

Your mission is to design a model-routing policy using measured task outcomes.

You are given a repository with sample agent tasks that vary by risk, ambiguity, and verification cost.

The duration for this challenge is 30 min or less.

## Project

[model-routing-app](./model-routing-app) contains the model routing workflow for this exercise.

## How To Go About It

Use [OpenAI Evals](https://github.com/openai/evals) or an equivalent eval harness to compare routing choices.

Ask your coding agent to inspect `model-routing-app/`, create the routing eval, propose the policy, and verify it on sample tasks.

## Evidence

Produce the eval cases, results, routing policy, and verification notes.

Raise the completed work as a PR for getting verified with our team.
