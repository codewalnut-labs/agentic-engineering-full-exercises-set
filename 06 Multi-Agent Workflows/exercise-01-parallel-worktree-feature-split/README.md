# Exercise 01 : Parallel Worktree Feature Split

## Your Mission

Your mission is to split independent feature work across isolated worktrees and integrate the results.

You are given a repository with three small improvements that should not block each other.

The duration for this challenge is 30 min or less.

## Project

[worktree-feature-app](./worktree-feature-app) contains the feature split workflow for this exercise.

## How To Go About It

Use [Git worktrees](https://git-scm.com/docs/git-worktree) for parallel agent work.

Ask your coding agent to inspect `worktree-feature-app/`, divide the work, run each slice independently, integrate the changes, and verify the app.

## Evidence

Produce the integrated change, ownership map, worktree log, and verification output.

Raise the completed work as a PR for getting verified with our team.
