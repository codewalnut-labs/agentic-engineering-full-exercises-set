# Exercise 02 : Subagent NFR Swarm

## Your Mission

Your mission is to use specialist agent passes for security, accessibility, performance, and testability.

You are given a repository with a feature that looks complete but has hidden non-functional risk.

The duration for this challenge is 30 min or less.

## Project

[nfr-swarm-app](./nfr-swarm-app) contains the NFR swarm workflow for this exercise.

## How To Go About It

Use [Superpowers](https://github.com/obra/superpowers) planning/review skills or equivalent subagent instructions.

Ask your coding agent to inspect `nfr-swarm-app/`, run focused specialist reviews, choose fixes, implement them, and verify the result.

## Evidence

Produce the specialist findings, selected fixes, implementation change, and verification output.

Raise the completed work as a PR for getting verified with our team.
