# Exercise 04 : Kanban Worktree Control Plane

## Your Mission

Your mission is to turn a noisy backlog into agent-ready cards and isolated implementation lanes.

You are given a repository with vague tasks, shared files, and unclear merge order.

The duration for this challenge is 30 min or less.

## Project

[kanban-control-app](./kanban-control-app) contains the control-plane workflow for this exercise.

## How To Go About It

Use Git worktrees and a small Kanban control plane for ownership, review, and integration.

Ask your coding agent to inspect `kanban-control-app/`, write agent-ready cards, assign isolation boundaries, complete one card, and verify it.

## Evidence

Produce the cards, ownership map, completed change, and verification output.

Raise the completed work as a PR for getting verified with our team.
