# Exercise 02 : Superpowers Feature Implementation

## Your Mission

Your mission is to implement a feature using a structured agent workflow.

You are given a repository that contains a feature request. Use Superpowers skills to plan and implement the feature.

The duration for this challenge is 60 min or less.

## Project

[team-collaboration-app](./team-collaboration-app) contains the feature requirement for this exercise.

## How To Go About It

Use [Superpowers](https://github.com/obra/superpowers) skills on the provided feature request.

Ask your coding agent to use Superpowers workflow to complete the feature implementation.

## Evidence

Produce the implemented feature code and Superpowers generated markdown artifacts.

Raise the completed work as a PR for getting verified with our team.
