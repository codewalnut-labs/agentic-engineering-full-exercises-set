# Exercise 05 : Graphiti Code Review Graph Skill

## Your Mission

Your mission is to review a cross-cutting change using graph context instead of a flat diff.

You are given a repository where a discount change affects call paths, ownership boundaries, and tests.

The duration for this challenge is 30 min or less.

## Project

[review-graph-app](./review-graph-app) contains the graph review workflow for this exercise.

## How To Go About It

Use [Graphiti](https://www.getzep.com/platform/graphiti/) or an equivalent review graph workflow.

Ask your coding agent to inspect `review-graph-app/`, create the review graph, find risky paths, fix blockers, and verify behavior.

## Evidence

Produce the graph notes, review findings, fixes, and verification output.

Raise the completed work as a PR for getting verified with our team.
