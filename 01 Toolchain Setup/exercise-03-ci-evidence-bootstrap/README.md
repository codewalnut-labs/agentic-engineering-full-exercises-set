**Exercise 03**

# Agent Check Bootstrap

**Goal:** Turn a fragile manual setup into a repeatable agent-safe local gate with rules, hooks, allowed tools, and clear verification commands.

**Outcome:** The starter no longer depends on manual verification; CI can produce a reviewable evidence bundle.

## Start Here

Starter folders:
- [starter-react](./starter-react)

React starter (run from the repository root):

```bash
cd "01 Toolchain Setup/exercise-03-ci-evidence-bootstrap/starter-react"
npm install
npm run dev
```

- [docs/ci-plan.md](./docs/ci-plan.md)

## Do This

1. Run the starter and skim the seed files so the agent has real context.
2. Ask your coding agent for a short plan that names files, checks, and risks before it edits.
3. Add a local `agent:check` toolchain command that runs install validation, lint/format/typecheck/build, and guardrail simulation.
4. Add or update AGENTS.md/CLAUDE.md rules so the agent knows safe commands, forbidden paths, and the verification gate.
5. Implement deny rules or a PreToolUse-style hook simulator for secrets, destructive git commands, and unapproved CLIs.
6. Codify one repeated correction as a durable rule and prove the agent gate catches it.
7. Run the checks below and keep the output for your evidence note.
8. Commit only the files needed for this exercise.

## Deliver

- CI workflow or `scripts/ci-check` that runs locally.
- Evidence artifact generator with real command output.
- At least one app or test change needed to make the gate meaningful.
- A PR evidence summary generated by the script.

## Verify

Run at least:

```bash
cd "01 Toolchain Setup/exercise-03-ci-evidence-bootstrap/starter-react" && npm test
cd "01 Toolchain Setup/exercise-03-ci-evidence-bootstrap/starter-react" && npm run agent:check
```

Done when:
- agent:check
- rules command audit
- hook simulator
- CLI/MCP allowlist check
- A short evidence note lists commands run, pass/fail results, changed behavior, and residual risk.

A README-only answer is not enough; the exercise is complete only when the working change and evidence are in place.
