# AGENTS.md

This repository is an exercise source. Learner solutions should stay inside the exercise folder they picked.

## Commands

- React starters: `npm install`, `npm run dev`, `npm run build`, and when present `npm test` or `npm run test:e2e`.
- Spring Boot starters: `mvn test` and `mvn spring-boot:run`.

## Conventions

- Prefer React + TypeScript for UI work.
- Use Java + Spring Boot only for exercises that need backend contracts or integration tests.
- Keep generated evidence, specs, diagrams, reviews, and retros in the selected exercise folder.
- Do not add secrets, tokens, or real customer data.

## Guardrails

- Do not edit unrelated exercise folders while solving one exercise.
- Do not read or create real `.env` files. Use documented example values.
- Do not use destructive git commands as part of an exercise solution.
- Record commands and verification results beside the implemented solution and evidence gate.
