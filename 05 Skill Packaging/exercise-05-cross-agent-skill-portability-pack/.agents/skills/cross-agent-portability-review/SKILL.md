---
name: cross-agent-portability-review
description: Review an agent skill for portable paths, precise triggers, scoped tools, and progressive-disclosure references.
---

Use this skill before committing a team skill that should work across multiple coding agents.

Checklist:
1. Reject absolute machine paths and personal account references.
2. Keep description trigger words concrete and bounded.
3. Move long checklists into references loaded only when needed.
4. Document tool assumptions instead of requesting every tool.
5. Produce an install matrix and remaining portability risks.
