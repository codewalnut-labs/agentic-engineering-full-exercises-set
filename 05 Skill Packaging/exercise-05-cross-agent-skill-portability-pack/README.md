# Exercise 05 : Cross-Agent Skill Portability Pack

## Your Mission

Your mission is to make one workflow portable across multiple coding-agent skill formats.

You are given a repository with a compliance review workflow that currently works only as a chat prompt.

The duration for this challenge is 30 min or less.

## Project

[skill-portability-app](./skill-portability-app) contains the portability workflow for this exercise.

## How To Go About It

Use the common [Agent Skills](https://github.blog/changelog/2025-12-18-github-copilot-now-supports-agent-skills/) pattern and adapt it for your available agents.

Ask your coding agent to inspect `skill-portability-app/`, package the workflow for at least two agent formats, run it once, and verify the output.

## Evidence

Produce the portable skill files, run output, and verification notes.

Raise the completed work as a PR for getting verified with our team.
