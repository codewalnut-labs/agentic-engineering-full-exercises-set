# Exercise 02 : Release Notes Skill Factory

## Your Mission

Your mission is to turn messy release-note work into a reusable skill with consistent output.

You are given a repository with changes, partial evidence, and unclear release-note grouping.

The duration for this challenge is 30 min or less.

## Project

[release-notes-app](./release-notes-app) contains the release-note workflow for this exercise.

## How To Go About It

Use [Claude Skills](https://support.claude.com/en/articles/12512176-what-are-skills) or an equivalent agent skill package.

Ask your coding agent to inspect `release-notes-app/`, create the release-note skill, run it, and verify the generated notes against the diff.

## Evidence

Produce the skill package, generated release notes, and verification notes.

Raise the completed work as a PR for getting verified with our team.
