---
name: skill-trigger-eval-harness
description: Evaluate agent skills for trigger precision, required process steps, output schema, and regression behavior.
---

Use this skill when a team skill needs trigger tests, regression checks, or a measurable improvement loop.

Workflow:
1. Read the candidate SKILL.md and list its intended trigger conditions.
2. Create positive, negative, and ambiguous prompts before editing the skill.
3. Score process behavior, output shape, file hygiene, and efficiency.
4. Patch skill metadata and instructions only after the failing cases are clear.
5. Produce a versioned report with pass/fail results and remaining risk.
