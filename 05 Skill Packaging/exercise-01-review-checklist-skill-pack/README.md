# Exercise 01 : Copilot Review Skill Pack

## Your Mission

Your mission is to package a repeated review workflow into a reusable agent skill.

You are given a repository with a risky app change and scattered review expectations.

The duration for this challenge is 30 min or less.

## Project

[review-skill-app](./review-skill-app) contains the review workflow for this exercise.

## How To Go About It

Use [GitHub Copilot Agent Skills](https://github.blog/changelog/2025-12-18-github-copilot-now-supports-agent-skills/) or the same skill structure in your coding agent.

Ask your coding agent to inspect `review-skill-app/`, package the review checklist skill, run it on the change, and verify the findings.

## Evidence

Produce the skill package, review output, and verification notes.

Raise the completed work as a PR for getting verified with our team.
