# Exercise 01 : GitHub Actions PR Evidence Pack

## Your Mission

Your mission is to automate a PR evidence pack that reviewers can trust.

You are given a repository where PRs often miss test output, screenshots, risks, and rollback notes.

The duration for this challenge is 30 min or less.

## Project

[pr-evidence-app](./pr-evidence-app) contains the PR evidence workflow for this exercise.

## How To Go About It

Use [GitHub Actions](https://docs.github.com/actions) to gather repeatable PR evidence.

Ask your coding agent to inspect `pr-evidence-app/`, add the evidence workflow, make one small app change, and verify the generated artifacts.

## Evidence

Produce the workflow, evidence artifact, app change, and verification output.

Raise the completed work as a PR for getting verified with our team.
