# Skill and Hook Improvement Loop

## Use When

- The task matches this exercise mission.
- The agent needs a repeatable checklist rather than another ad-hoc prompt.

## Do Not Use When

- The learner has not yet read the exercise README.
- The requested output is outside this exercise folder.

## Workflow

1. Read the exercise README and starter docs.
2. Inspect only the files needed for the current phase.
3. Produce the requested retrospective output with evidence and open questions.
4. Record what the skill missed so it can be improved.
