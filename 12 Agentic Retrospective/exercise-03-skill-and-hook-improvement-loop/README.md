# Exercise 03 : Superpowers Skill Improvement Loop

## Your Mission

Your mission is to improve a flawed skill and hook setup using evidence from a failed run.

You are given a repository where a skill triggers too broadly and a hook catches problems too late.

The duration for this challenge is 30 min or less.

## Project

[skill-hook-app](./skill-hook-app) contains the skill improvement workflow for this exercise.

## How To Go About It

Use [Superpowers](https://github.com/obra/superpowers) or an equivalent skill improvement workflow.

Ask your coding agent to inspect `skill-hook-app/`, analyze the failed run, improve the skill and hook, and verify the new behavior.

## Evidence

Produce the improved skill, hook change, before/after notes, and verification output.

Raise the completed work as a PR for getting verified with our team.
