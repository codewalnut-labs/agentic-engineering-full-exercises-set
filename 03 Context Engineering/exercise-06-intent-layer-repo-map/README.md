# Exercise 06 : Intent Layer Repo Map

## Your Mission

Your mission is to add folder-level intent context so future agents understand ownership before editing.

You are given a repository with repeated agent mistakes caused by unclear boundaries and hidden conventions.

The duration for this challenge is 30 min or less.

## Project

[incident-context-app](./incident-context-app) contains the incident workflow for this exercise.

## How To Go About It

Use [Serena](https://github.com/oraios/serena) or an equivalent symbol-aware repository mapping workflow.

Ask your coding agent to inspect `incident-context-app/`, create the intent map, make one safe workflow improvement, and verify it.

## Evidence

Produce the intent map, the workflow change, and verification output.

Raise the completed work as a PR for getting verified with our team.
