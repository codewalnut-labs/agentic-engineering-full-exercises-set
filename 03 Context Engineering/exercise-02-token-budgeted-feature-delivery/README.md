# Exercise 02 : Serena Token-Budgeted Feature Delivery

## Your Mission

Your mission is to deliver a small feature while keeping the agent context intentionally small.

You are given a repository that contains account-health logic spread across UI, domain, and test files. Use semantic code navigation before loading broad file context.

The duration for this challenge is 30 min or less.

## Project

[account-health-app](./account-health-app) contains the account-health feature for this exercise.

## How To Go About It

Use [Serena](https://oraios.github.io/serena/01-about/000_intro.html) or an equivalent semantic code navigation workflow.

Ask your coding agent to inspect `account-health-app/`, record which symbols and files enter context, implement the feature, and verify it.

## Evidence

Produce the feature change, tests, and a token-budget context log.

Raise the completed work as a PR for getting verified with our team.
