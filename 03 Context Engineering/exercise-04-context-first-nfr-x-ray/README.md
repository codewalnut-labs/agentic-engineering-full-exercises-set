# Exercise 04 : Graphiti NFR Context Memory

## Your Mission

Your mission is to create reusable context memory before asking an agent to audit non-functional requirements.

You are given a repository with performance, accessibility, and security hints scattered through code and docs. Build memory first, then act on the highest-value finding.

The duration for this challenge is 30 min or less.

## Project

[nfr-audit-app](./nfr-audit-app) contains the NFR audit workflow for this exercise.

## How To Go About It

Use [Graphiti](https://github.com/getzep/graphiti) or an equivalent temporal knowledge-graph workflow.

Ask your coding agent to inspect `nfr-audit-app/`, capture durable NFR context, fix one priority issue, and verify it.

## Evidence

Produce the context memory notes, the fix, and the verification output.

Raise the completed work as a PR for getting verified with our team.
